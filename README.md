# Python爬虫新闻分析系统 python431

## 项目概述

本项目是一个基于Python、Django和Vue.js的前后端分离的新闻分析系统。该系统不仅提供基本的新闻浏览与评论功能，还能对新闻进行分类管理和数据分析。

## 技术栈

- 前端：Vue.js
- 后端：Python、Django、Scrapy
- 数据库：Mysql

## 功能模块

### 用户功能

- 我的收藏
- 个人信息管理
- 登录注册
- 查看热点新闻
- 查看新闻资讯
- 新闻评分
- 新闻评论

### 管理员功能

- 用户管理
- 新闻分类管理
- 新闻资讯管理
- 热点新闻管理
- 新闻评分管理
- 主题标签管理
- 数据可视化

## 系统角色

- 普通用户：可以查看、评论、评分新闻，管理个人收藏和个人信息。
- 管理员：拥有用户管理权限，可进行新闻的分类管理、资讯管理和系统数据可视化等操作。

## 运行环境

- 后端：Python 3+、Django 2.x、Scrapy 1.x
- 前端：Vue.js 2.x
- 数据库：Mysql 5.x 或更高版本

## 部署步骤

（以下步骤为简化版，具体部署需根据实际环境进行调整）

1. 环境准备：安装Python 3、Mysql数据库和所需Python库。
2. 数据库配置：根据提供的SQL文件创建数据库和表。
3. 后端部署：配置Django项目和Scrapy爬虫。
4. 前端部署：编译Vue.js项目并配置静态文件。
5. 系统测试：启动服务进行功能测试。

- 前后端分离设计，提高开发效率和维护便捷性。
